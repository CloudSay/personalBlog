## 机器学习基础

机器学习是人工智能的核心，本章节将介绍机器学习的基本概念、常用算法和实践方法。

### 监督学习
- 线性回归
- 逻辑回归
- 决策树
- 支持向量机
- 随机森林

### 无监督学习
- 聚类算法（K-means、层次聚类）
- 降维算法（PCA、t-SNE）

### 强化学习
- 马尔可夫决策过程
- Q学习
- 策略梯度

## 深度学习

深度学习是机器学习的一个分支，基于神经网络进行学习。

### 神经网络基础
- 感知机
- 激活函数
- 损失函数
- 反向传播
- 梯度下降

### 卷积神经网络（CNN）
- 卷积层
- 池化层
- 经典网络架构（LeNet、AlexNet、VGG、ResNet）

### 循环神经网络（RNN）
- LSTM
- GRU
- 序列模型

### Transformer
- 自注意力机制
- 编码器-解码器结构
- BERT
- GPT

## 自然语言处理

自然语言处理（NLP）是让计算机理解和生成人类语言的技术。

### 文本预处理
- 分词
- 停用词过滤
- 词向量化（Word2Vec、GloVe、TF-IDF）

### 经典任务
- 文本分类
- 情感分析
- 命名实体识别
- 机器翻译
- 文本生成

## 计算机视觉

计算机视觉让计算机理解和分析图像与视频。

### 经典任务
- 图像分类
- 目标检测
- 图像分割
- 图像生成

### 热门技术
- 迁移学习
- 数据增强
- 生成对抗网络（GAN）
- 扩散模型
